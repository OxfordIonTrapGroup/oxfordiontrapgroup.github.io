Oxford Ion Trap Quantum Computing group
Open-source projects

Much of the software and hardware developed as part of our work is open source.

This index showcases some of our own projects that are not specific to our group; we are also regular contributors to the ARTIQ/Sinara ecosystem.

Feel free to be in touch if any of the projects look like they might be useful in your own laboratory as well!

Clarendon Laboratory, Oxford

entangler-core

Pulse sequencing and heralding logic for remote entanglement generation, implemented as an FPGA state machine for minimum latency.

Python (Migen)

ion_phys

A nascent toolkit for calculating state energies, transition matrix elements, rate equations, solving OBEs, etc., using Python/NumPy.

Python

mjolnir (Thor's Camera)

Network-capable laser beam profiling user interface for Thorlabs (and IDS) cameras.

Python

ndscan

ARTIQ framework for composing experiments from reusable building blocks, and interactive exploration through one- or multidimensional scans.

Python

oitg

Common Python libraries not directly dependent on ARTIQ, e.g. for result data loading and fitting, or generating RBM/GST/… circuits.

Python

oxart-devices

ARTIQ/SiPyCo-compatible device drivers for various laboratory devices (power meters, linear/piezo stages, laser controllers, …).

Python

Oven

Reproducible mechanical design for electrically-heated thermal atom sources, and electronics/software for fast-response closed-loop control.

SolidWorks, Altium, C, Python

Squareatron-5000

RF limiting amplifier with low phase-/amplitude-modulation noise, designed for stabilising ion trap RF drive voltages before amplification.

Altium

stabilizer_curent_sense

Low-noise, high-stability current sensing and modulation frontend for the Stabilizer open-hardware control platform (e.g. for magnetic field coils).

Altium, LTSpice, IPython

WAnD

Network-based cw laser diagnostics suite for HighFinesse wavemeters (+optional OSAs). Supports multiple servers, logging, feedback locking, etc.

Python
